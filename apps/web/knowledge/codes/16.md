## Common defences
- Valid permit displayed but obscured (similar to code 12).
- Virtual permit registered against VRM.
- Bay sign / markings unclear so the permit-only nature wasn't apparent.

## Common rebuttals from council
- "Permit not visible" — counter with dashboard photo.
- "Bay clearly marked permit only" — counter with photo of the bay condition.

## Notes
Code 16 is functionally very similar to 12 — the difference is the
bay class (16 is generic permit bay, 12 is residents'/shared use).
Same defence framework applies.
