# Managing councils

> :material-pencil-outline: **Stub.** Filled when Phase B's admin UI ships.

The admin UI exposes CRUD operations against the [council knowledge base schema](../architecture/knowledge-base.md):

- **List view** with filters on `type`, `automation_status`, `last_verified_at`.
- **Detail/edit** view for each council record.
- **Bulk verify** workflow.
- **Form-schema editor** with live JSON validation (used by v0.2 automated submission).
- **Audit log** showing the last 10 changes per record.
